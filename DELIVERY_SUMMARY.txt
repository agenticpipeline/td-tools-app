================================================================================
                          ESTIMATION ENGINE FEATURES
================================================================================

✓ Threshold-based crew sizing (1-4 items=2, 5-12=4, 13-24=6, 25-48=8, 49+=ceil/6)
✓ Batch repetition scaling: installTime = baseTime + (qty-1) × baseTime × 0.6
✓ OSHA weight triggers (>50kg = minimum 4 crew)
✓ Flown/rigged items detection (minimum 4 crew)
✓ High complexity scaling (≥4.0 average = crew × 1.25)
✓ Separate install and dismantle calculations
✓ Schedule feasibility checking
✓ Department-specific tracking (Audio, Video, Lighting, Scenic, Softgoods, Stagehands)
✓ Cost calculation ($65/hour × crew × hours)
✓ Risk flag generation (RIGGING_REQUIRED, HEAVY_EQUIPMENT, HIGH_COMPLEXITY, etc.)
